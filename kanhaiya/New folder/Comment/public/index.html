This feature of the app is called CHORUS. The feature is a music video competition. This feature is basically for users that are musically gifted to upload videos to be voted into CHORUS, an online singing competition. Voting features will be available for users. Followers and people followed can use the buttons provided to vote for the singer to stand a chance of getting into the singing competition on 4U2NV’s CHORUS music competition. Alternatively, there will be a link a user can click to send he/her voting link to any 4U2NV profile user to vote. (click link – space will upload to search for name, list of all followers will also upload - type name – send link to the desired name requesting them to vote for you – the receiver clicks accept or decline. When the receiver clicks accept, the vote will automatically be recorded).
